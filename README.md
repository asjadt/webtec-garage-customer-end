# Garage Management System - Customer End

A customer-facing interface for managing garage-related services, built using modern web technologies.

## Table of Contents

- [Technologies Used](#technologies-used)
- [Installation](#installation)
- [Usage](#usage)

## Technologies Used

This project leverages the following technologies and frameworks:

- **Frontend Framework**: React (18.2.0)
- **Styling**: Tailwind CSS (3.3.2), DaisyUI
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM
- **Form Management**: React Hook Form
- **Testing**: Cypress, Vitest, Testing Library
- **Maps**: Google Maps API
- **PDF/Document Handling**: React-PDF, jsPDF, @react-google-maps/api
- **Charts**: Chart.js, React-Chartjs-2
- **Bundler**: Vite
- **Other Utilities**: Axios, Day.js, Moment.js, D3.js, Framer Motion

## Installation

Follow these steps to set up the project locally:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/asjadt/webtec-garage-customer-end.git
   cd webtec-garage-customer-end
   ```

2. **Install Dependencies: Using npm:**
    ```bash
     npm install
    ```
    or
    ```bash
     yarn install
    ```

4. **Run the Development Server:** Start the development server to preview the application locally.
   ```bash
    npm run dev
   ```
   or
   ```bash
   yarn dev
   ```

6. **Build for Production: To create a production-ready build, run:**
   ```bash
   npm run build
   ```
   or
   ```bash
   yarn build
   ```
   

### Usage
After starting the development server, navigate to http://localhost:5172 in your browser to access the application. Configure additional settings and integrations based on your specific Garage management system customer end needs.
